Processing

Moving Circles
This is the first sketch I have created with processing. It includes a sine wave as well as while and if statements. The while statements controls the points in which the sine waves are set at. I alternated the lengths of the waves. The if statements controls the three rows of random flashing colored ellipses that repeats on the opposite side of the border when hitting the other.
Chaos
In the first example I learned how to condense arrays and later define them in an efficient way. I used low and high arguments in my speedX and speedY variables to generate random speeds. I used if statements to reset my blocX, blocY, speedX, and speedY values to bounce off the edges of the workspace. There are 20 balls that are set to move and generate random colors that leave a lasting trail behind. I also included a diagonal set of rectangles to flash using the millis command.
