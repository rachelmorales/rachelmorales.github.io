Drawing Tool

For my drawing tool project I have created a sweet machine consisting of 5 drawing tools. Atoms

For this sketch the algorithm that I chose to focus on was Perlin Noise. I created two custom classes: Atom and Noiseloop. In the Atom class I defined the position, diameter, and red and blue color values of a circle shape. In the Noiseloop class I randomized those values that I declared in the Atom class.
